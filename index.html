Projects

Automated Renewal Reminder Email System Project Description: This project automates the process of sending renewal reminder emails to members of the KSPCA (Kenya Society for the Protection & Care of Animals). The script checks renewal dates from an Excel sheet and sends personalized reminder emails to members whose subscriptions are expiring or have expired. Key Features: Automated Email Generation: The script automatically generates and sends reminder emails through Microsoft Outlook. Personalized Messages: Each email is personalized with the member's first name, last name, and specific renewal details. Subscription Management: The script checks for subscription expiry and sends different emails based on whether the subscription is expiring today or has already expired. Technical Details: VBA Programming: Utilized VBA within Excel for automation. Microsoft Outlook Integration: Integrated with Outlook on Mac to send emails. Date Calculations: Automatically calculates expiry dates and constructs appropriate messages. Project Highlights: Automated email generation ensures timely reminders, enhancing member retention. Personalized communication fosters a strong relationship with members. Seamless integration with Microsoft Outlook makes the process efficient and reliable.
